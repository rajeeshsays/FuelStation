# **App Name**: FMS Fuel Station Manager

## Core Features:

- Price Display: Capture and display current fuel prices for each grade (Lanka Auto Diesel, Lanka Petrol, Kerosene). Reflect CEYPETCO standards.
- Daily Meter Reading & Sales Calculation: Input daily meter readings for each pump (3 Lanka Auto Diesel, 2 Lanka Petrol, 1 Kerosene). Calculate daily sales by subtracting previous day's reading.
- Stock Incoming Logging: Record stock incoming with fields for invoice number, invoice date, fuel type, price per liter, number of liters, discount, final price, and invoice photo attachment.
- Stock Incoming Viewing & Filtering: View stock incoming data, filterable by fuel type, date, and other parameters.
- Inventory Tracking: Track inventory by subtracting fuel sales and adding new stock. Viewable by date and fuel type.
- Daily Sales Reports: Generate daily sales reports by fuel type and date, broken down by pump.
- Sales Reporting (Daily, Monthly, Yearly): Generate monthly, daily, and yearly sales reports by fuel type, filterable by date ranges.

## Style Guidelines:

- Primary color: Red, reflecting the CEYPETCO brand and creating a sense of urgency.
- Secondary color: Light Gray, providing a neutral background that balances the red and reduces visual fatigue.
- Tertiary color: Off-White, used for text and data displays to ensure readability against the gray background.
- Accent color: White, used sparingly for key interactive elements and highlights.
- Font: Use a clean, readable sans-serif font suitable for displaying numerical data and text in English and Sinhala/Tamil (if needed).
- Use clear and recognizable icons to represent different fuel types, data points, and actions.
- Tab-based navigation for the main functions: Daily Meter Reading, Stock Incoming, Inventory Tracking, Daily Reports, Sales Reporting. Ensure a clear and structured layout for efficient data entry and viewing.
- Subtle animations for loading states and data updates to provide feedback to the user.
